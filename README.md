# Customer_Behavior_Analysis
Customer Behaviour Analysis project using SQL, Python, Jupyter Notebook, and Power BI to analyze customer purchasing patterns, sales trends, and business insights. Performed data cleaning, EDA, SQL queries, and created interactive dashboards for visualization and decision-making.
# Customer Behaviour Analysis

## Overview
The Customer Behaviour Analysis project is designed to analyze customer purchasing patterns and generate business insights using SQL, Python, and Power BI. The project helps in understanding customer preferences, sales performance, and behavioral trends through data analysis and visualization.

## Technologies Used
- Python
- Jupyter Notebook
- SQL
- Power BI
- Pandas
- NumPy
- Matplotlib
## Project Workflow

### 1. Data Collection
- Collected customer transaction and sales data.

### 2. Data Cleaning
- Handled missing values
- Removed duplicates
- Formatted and cleaned data using Python libraries

### 3. SQL Analysis
Performed SQL queries for:
- Customer segmentation
- Sales analysis
- Revenue calculation
- Product performance analysis
- Customer purchase frequency

### 4. Exploratory Data Analysis (EDA)
Used Python for:
- Data visualization
- Trend analysis
- Customer behavior insights
- Finding correlations and patterns

### 5. Dashboard Creation
Created interactive dashboards in Power BI to visualize:
- Sales trends
- Customer insights
- Revenue performance
- Product analysis
## Features
- Customer purchase analysis
- Sales trend visualization
- Customer segmentation
- Interactive Power BI dashboard
- Business insight generation
